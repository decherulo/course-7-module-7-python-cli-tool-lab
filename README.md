# Project Management CLI Tool

A command-line interface (CLI) application built with Python and SQLAlchemy to manage projects, tasks, and users efficiently.

---

## Features

- **Project Management:** Create, view, update, and delete projects.
- **Task Tracking:** Assign tasks to specific projects and update their completion status.
- **Data Persistence:** Uses SQLite and SQLAlchemy ORM to safely store and retrieve data.
- **Interactive CLI:** User-friendly terminal interface for executing commands.

---
